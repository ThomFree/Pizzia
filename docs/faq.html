Qu'est-ce que Pizzia ?

Pizzia est une API vous permettant de créer des modules pour votre Zia facilement. En respectant les principes de l'API Pizzia lors de la réalisation de vos modules, vous vous assurez de réaliser quelque chose de cohérant, fournissant un haut niveau d'abstraction et permettant à votre module de s'occuper uniquement de sa tâche, sans se soucier du reste du serveur HTTP.

Comme pour le principe algorithmique du divide and conquer, pour diminuer la compléxité de la réalisation d'un serveur web, le découpage en plusieurs modules réalisant des micro-tâches est essentiel. Pizzia vous permet donc de réaliser les différents modules en respectant l'interface de programmation que nous vous proposons. Retrouvez la suite de ces explications dans le quickstart.

Pourquoi voter pour Pizzia lors de l'élection ?

Afin de pouvoir utiliser Pizzia pour votre Zia, nous devons être élu. Voici de nombreuses bonnes raisons d'opter pour Pizzia plutôt qu'une autre API :
  • Pizzia offre une API avec un haut niveau d'abstraction, ce qui vous promet d'être hautement productif ainsi qu'un respect total de la RFC2616
  • Pizzia est documenté de manière complète, vous pourrez retrouver sur notre repository ainsi que sur notre site de nombreux schémas et documentations ; ici-même vous avez à votre disposition un quickstart pour vous initier à la philosophie de l'API, un guide complet qui vous explique à la manière d'un tuto comment implémenter chacune des interfaces fournies par Pizzia et créer votre premier module, des exemples de modules déjà conçus implémentant l'API Pizzia afin de vous permettre d'apréhender plus facilement l'API, et enfin un Doxygen pour retrouver la documentation de chaque fonction offerte par notre API.
    En bref, nous avons produit un nombre conséquent de documents afin de vous aider au maximum à comprendre Pizzia !
  • Pizzia a été conçu et pensé afin de vous laisser la possibilité de respecter de nombreux design patterns pour offrir une façon moderne et solide de réaliser vos modules. En implémentant notre API, vous allez pouvoir réaliser votre serveur web à la façon d'une pipeline (gérée par un mediator par exemple).
  • Nous offrons un support via Telegram ou Discord (mais aussi IRL ! =D) pour vous aider à implémenter l'API, si vous avez des problèmes de compréhensions par exemple ou qu'un point est flou, nous nous ferons un plaisir d'échanger avec vous.
  • Nous permettons à tous de contribuer à notre API librement via GitHub, ce qui vous permet vous aussi si vous le souhaitez de pouvoir apporter votre pierre à l'édifice et de devenir contributeur de Pizzia. Pour cela, il vous suffit de vous référer à la partie plus bas de la FAQ qui explique comment contribuer.
